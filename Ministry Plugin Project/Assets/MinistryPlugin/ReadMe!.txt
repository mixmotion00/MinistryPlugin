1. From top Menu > Edit > Project Setting > Script Execution Order
2. Simply drag MinistrySplashScreen.cs to the Script Execution Editor editor


HOW TO USE THIS PLUGIN
1. Right click from the hierachy area, Create Other > Ministry > Ministry Splash Screen[Mobile]
2. Drag the PNG(Sprite file) to SplashScreen2(if only need second splash screen)

OPTIONAL ***
The logo spawn time can be change with Splash Logo Time variable.
